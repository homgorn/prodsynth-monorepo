# ProdSynth Licensing & IP (BMAD v2.0 — Phase 4)

## 1. License Compatibility Engine

### The Problem
Users paste GitHub repos with ANY license:
- **MIT, Apache 2.0, BSD** → ✅ Compatible
- **GPL 2/3, AGPL** → ⚠️ Contamination risk
- **CC (non-commercial, no-derivs)** → ❌ Blocked
- **No license** → ❌ Blocked by default

### License Checker Agent

```python
# packages/agents/research.py → LicenseChecker

COMPATIBLE = {"MIT", "Apache-2.0", "BSD-2-Clause", "BSD-3-Clause", "ISC", "Unlicense"}
CONTAMINATING = {"GPL-2.0", "GPL-3.0", "AGPL-3.0", "LGPL-2.1", "LGPL-3.0"}
BLOCKED = {"CC-BY-NC", "CC-BY-NC-ND", "CC-BY-ND", "Proprietary"}

def check_license(repo_path: str) -> LicenseReport:
    license_file = find_license_file(repo_path)  # LICENSE, LICENSE.txt, COPYING, etc.
    if not license_file:
        return LicenseReport(
            license="NONE",
            risk="HIGH",
            message="No license found. You cannot use or distribute this code legally.",
            action="BLOCK"
        )

    license_name = parse_spdx(license_file)
    if license_name in BLOCKED:
        return LicenseReport(
            license=license_name,
            risk="CRITICAL",
            message="Non-commercial or no-derivatives license detected.",
            action="BLOCK"
        )

    if license_name in CONTAMINATING:
        return LicenseReport(
            license=license_name,
            risk="MEDIUM",
            message="Copyleft license detected. Generated code may inherit GPL obligations.",
            action="WARN_CONTAMINATION"
        )

    return LicenseReport(
        license=license_name,
        risk="LOW",
        message=f"{license_name} is compatible with MIT distribution.",
        action="ALLOW"
    )
```

### Contamination Detection

When a GPL repo is detected:
1. **Warn user** — "This repo uses GPL. Generated code may require GPL licensing."
2. **Isolate analysis** — Don't use GPL code snippets as-is
3. **Substitute patterns** — Replace GPL-licensed patterns with compatible alternatives
4. **Document** — Include license notice in generated output

### Example Flow
```
User submits: https://github.com/example/gpl-web-app
  ↓
LicenseChecker: GPL-3.0 detected (Contaminating)
  ↓
Warning modal: "This repo uses GPL v3. Generated code may require GPL licensing.
  You can still proceed, but the output will include a GPL compatibility notice."
  ↓
CodeAgent: substitutes GPL-specific patterns (e.g., uses MIT-licensed equivalents)
  ↓
Output: code + LICENSE.compatibility file
```

## 2. IP Ownership & Rights

### User Rights Over Generated Code

**Core Principle:** *The user owns the code they generate. We have no rights to it.*

| Aspect | User's Code | Our Code |
|--------|------------|---------|
| **Input repo** | Owned by original authors | — |
| **Generated code** | User owns 100% | — |
| **Templates from Marketplace** | Licensed per template | User owns output |
| **Our platform code** | — | Owned by ProdSynth |
| **BMD framework** | — | Public domain |

### License Grant from User

When a user generates code, they grant us:
1. **Right to store** the code in their workspace (for their use)
2. **Right to cache** for Graphiti (anonymized, no PII)
3. **Right to use** aggregated insights (never individual code)

### Anonymization for Graphiti

```python

def anonymize_node(node: Dict) -> Dict:
```

## 4. Third-Party Dependencies

### Dependency Licensing

Generated code includes `package.json` / `requirements.txt` with:
- **Pinned versions** (not ranges) — reproducibility
- **License field** in each package
- **Automated license audit** during generation

```python
class DependencyChecker:
    def check_dependencies(self, deps: List[Package]) -> DependencyReport:
        issues = []
        for pkg in deps:
            license = get_package_license(pkg.name, pkg.version)
            if license in BLOCKED:
                issues.append({
                    "package": f"{pkg.name}@{pkg.version}",
                    "license": license,
                    "severity": "HIGH",
                    "suggestion": f"Replace with MIT-licensed alternative: {get_alternative(pkg.name)}"
                })
        return DependencyReport(issues=issues)
```

| moment.js (MIT) | date-fns (MIT) |

## 5. DMCA & Legal Safety

### Safe Harbor

We operate under DMCA Safe Harbor (17 U.S.C. § 512):
- We don't store user code permanently
- Code is transient (user-owned, deleted on request)
- We don't index for search

### Content Policy

```python
PROHIBITED_PATTERNS = [
    "cryptojacking",
    "adsware",
    "data exfiltration",
    "backdoor",
    "keylogger",
    "credential harvesting",
    "phishing",
    "scraping personal data without consent",
]

# SafetyGuard blocks generation if prohibited patterns detected
async def safety_check(code: str) -> SafetyResult:
    for pattern in PROHIBITED_PATTERNS:
        if match(pattern, code):
            return SafetyResult(
                status="BLOCKED",
                reason=f"Prohibited pattern detected: {pattern}",
                action="REPORT"
            )
    return SafetyResult(status="ALLOWED")
```

### Terms of Service Requirements

- [ ] They understand GPL contamination risk (if applicable)
